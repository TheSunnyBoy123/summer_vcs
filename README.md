# SOL VCS

# Commands

## `create`

## Synopsis

<pre>
<b>sol create</b> [-h|--help] [-w|--warning] [-v|--verbose] [-h|--help] <b>[<directory>\]</b>
</pre>

## Descriptions
Creates a sol repository in the given directory. Default directory is the current directory.
